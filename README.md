# -2025年秋学期プロジェクト

2025年秋学期のプロジェクトで、内容は基本的なウェブ開発とDeepL自動翻訳APIの導入を含む。
このプロジェクトでは、阿里雲のECSライトサーバーを使用し、Ubuntu 22.04（64ビット）に接続しています。ドメインからいつでもウェブサイトを確認できるよう、継続して更新していく予定です。

这玩意是2025年秋季学期的项目，上传到git和大家分享一下，其实就是个很基础的网站，接了一个DeepLy自动翻译而已，使用了阿里云的ESC轻量服务器，连接了Ubuntu 22.04 64位，我会保持续费以便随时可以通过域名查看网站。
